# Welcome to the Qantas.com EDS Storybook

Welcome to our interactive playground for components, plus useful tools to speed up your development workflow.

## How to Use Storybook

On the **left panel**, you'll find a list of all the components available in the system. Click on any component to see a preview and its usage details.

You can **modify the component's configuration** using the **Controls** tab. This allows you to interact with the components and change properties like color, size, and text dynamically.

Feel free to explore the components and start building!
